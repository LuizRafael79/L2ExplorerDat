# L2ExplorerDat Editor - 2.0

L2Explorer is a tool designed for opening and saving Lineage 2 .dat files.<br>
This software is intended to assist developers and players in managing, modifying,<br>
and understanding the data used by Lineage 2.

The app detects java version actually installed in Windows, Linux or MacOS

L2ExplorerDat.bat runs the app in Windows
L2ExplorerDat.sh runs the app in Linux/MacOS

## Features

- Open Lineage 2 .dat files.
- Save modifications back to .dat format.
- Mass Unpack and Repack
- Mass Patch of entire folder
- Change Encryption of files

## Recommended software

- Java JDK or Liberica JDK 25
- IntelliJ IDE for Java Developers
- Maven
